ZenFlow-Personal Productivity App! 

🎉 Here's what's packed in:

Dashboard — At-a-glance stats, today's tasks, project progress, and habit streaks
Tasks — Add tasks with priority (High/Medium/Low), assignee, due date, and reminder toggle
Calendar — Monthly view with tasks plotted on their due dates, color-coded by priority
Habits & Streaks — Tap to mark habits done daily, with streak counters and progress bars
Progress — Track project completion with +/- controls, plus task and habit completion stats
Dark/Light mode — Toggle with the 🌙/☀️ button
Reminder alerts — Due-today reminders pop up on the dashboard
